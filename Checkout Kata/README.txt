Checkout Kata
=============

You are working in a shop that needs a new till system. You want to sell:

Product Code		Product Name		Unit Price
A					Apples				£0.50
B					Bananas				£0.30
C					Cherries			£0.20
D					Dates				£0.10

You are now asked to add the following special offers:

Product				Offer
A					3 for £1.30
B					2 for £0.45

Now you need to remove the special offers above and add:

Product				Offer
C					3 for 2
D					50% off
